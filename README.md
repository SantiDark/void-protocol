# VOID PROTOCOL

A fast-paced infinite roguelike dungeon crawler built in pure HTML5 + Canvas.  
No frameworks. No dependencies. One file.

**Play it:** [santiagonegro.itch.io](https://santiagonegro.itch.io/)  
**Live build:** [santidark.github.io/void-protocol](https://santidark.github.io/void-protocol/)

---

## Gameplay

Descend through an endless, procedurally generated dungeon.  
Each floor is harder than the last. Each run is a new build.

- **WASD** — Move  
- **Mouse** — Aim & Shoot  
- **Space** — Dash (invincible frames)  
- **E** — Interact / Next floor  
- **Esc** — Pause  

---

## Features

- Infinite floors with procedural map generation
- 6 biomes with unique color themes
- Boss encounters every 5 floors
- 8 enemy types: Grunt, Runner, Shooter, Tank, Sniper, Bomber, Ghost, Elite
- 19 upgrades: Double Shot, Pierce, Explosive, Homing, Freeze, AOE Burst, Lifesteal and more
- Rarity tiers: Common / Rare / Epic — better odds deeper in the run
- Fullscreen + Pause menu

---

## Tech

Built with vanilla HTML5, CSS and JavaScript. Single `index.html` file.  
No build step. No dependencies. Drop it anywhere and it runs.
